# Issue: Integrate on-device LLM (MLC) for AI modules

**Description**

This task involves integrating the on-device Large Language Model (LLM) using the MLC framework. The goal is to enable AI-powered features within the app that can run locally without requiring a network connection.

**Acceptance Criteria**

- [ ] The MLC-based LLM is successfully integrated into the application.
- [ ] The AI modules (`AiTutorModule`, etc.) can use the local LLM for processing.
- [ ] The integration is tested for performance and stability on a target device.
- [ ] The app can gracefully handle cases where the LLM model is not yet downloaded or fails to load.

**Milestone**

v0.2: On-device Models
