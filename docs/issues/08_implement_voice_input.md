# Issue: Implement voice input feature for main modules

**Description**

To improve accessibility and user experience, this task is to implement a voice input feature. This would allow users to speak their prompts for features like AI chat or image generation instead of typing them.

**Acceptance Criteria**

- [ ] The app can capture audio input from the user's microphone (with permission).
- [ ] The captured audio is converted to text using Android's built-in speech-to-text capabilities or a third-party service.
- [ ] The transcribed text is used as input for the relevant module (e.g., as a chat message or an image generation prompt).
- [ ] The UI is updated to include a microphone icon or other control to initiate voice input.

**Milestone**

v0.4: Safety + UX
